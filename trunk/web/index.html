Version 1.2 released

15 May 2013

I've released version 1.2 of ForceDel. This is mainly a bugfix release for Windows XP users. Version 1.1 had a bug that caused ForceDel not to end in Windows XP.

Version 1.1 released

02 May 2013

I've released version 1.1 of ForceDel - now with support for Windows XP systems. On some systems ForceDel hung on exit - this issue is hopefully addressed, too.

Uploaded to SourceForge

26 February 2013

Since I've thoutht ForceDel could be of use for other people I decided to put it on SourceForge. I hope you'll find it useful and it will work for you as it works for me every day.

Please note that ForceDel does not run on Windows XP since it uses a Windows API that was introduced with Windows Vista and is therefore not available under Windows XP.
