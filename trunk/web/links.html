majala

majala is the short name for Managed Java Launcher, which is a professional exe wrapper for Java applications that supports console applications, graphical applications and Windows Services. majala is currently under development - but it is an just-for-fun project, so don't expect an release very soon.

PCL-Inspector

PCL-Inspector allows you to take a deep look into PCL5- and PCL-XL printer datastreams. The product is in an early design phase, so don't expect a release before 2014.

AFP-Inspector

AFP-Inspector allows you to take a deep look into AFP (MO:DCA) printer datastreams. The product is in an early design phase, so don't expect a release before 2014.

SmartBacker

SmartBacker is an online help utility - waked up by a hot-key, SmartBacker retrieves the information you need from its own database containing various manuals. The product is in an early design phase, so don't expect a release before 2014.
